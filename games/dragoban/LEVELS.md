# Dragoban Level Provenance

Dragoban embeds the complete 155-level **Microban I** collection by
David W. Skinner, in its original order. The source collection is kept
unchanged at `third-party-levels/david-w-skinner/microban.txt`.

The runtime representation preserves every wall, player, box, and goal.
It adds only empty exterior cells to fit maps into a `30 x 17` storage grid;
rendering uses each level's active bounds, so smaller maps retain large tiles.

Skinner permitted his sets to be freely distributed when kept unchanged and
credited with his name and email address. See `../../THIRD_PARTY_NOTICES.md`.

Legend:

```text
# wall
@ player
$ box
. goal
* box on goal
+ player on goal
space floor
```
